# telegram_contacts


הורידו את המאגר
```
git clone https://github.com/nhman-python/telegram_contacts.git
```

התקינו את הספריות
```
pip install -r requirements.txt
```

שנו את השורות 12 13 לערכים שלכם

bot_token = ''  # טוקן מפאדר בוט
chat_owner = 12344  # החלף בid שלך

הריצו את האתר
```
python3 contacts.py
```

או בווינדוס
```
python contacts.py
```
